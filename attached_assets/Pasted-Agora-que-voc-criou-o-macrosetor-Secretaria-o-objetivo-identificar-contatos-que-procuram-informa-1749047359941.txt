Agora que você criou o macrosetor Secretaria, o objetivo é identificar contatos que procuram informações administrativas e acadêmicas, como:

Documentação

Declarações

Certificados

Histórico

Mudança de dados cadastrais

Emissão de boletos para matrícula ou rematrícula (diferente de cobrança de inadimplência)

📌 Observações
Embora algumas dessas palavras também possam aparecer em cobranças ou suporte, o contexto acadêmico e administrativo define o macrosetor como Secretaria.

Idealmente, combine essa lógica com análise de canal (ex: número da Secretaria) e regras de precedência.

function identificarMacrosetor(texto: string): 'comercial' | 'suporte' | 'cobranca' | 'secretaria' {
  const t = texto.toLowerCase();

  const palavrasSecretaria = [
    'declaração', 'histórico', 'certificado', 'diploma', 'documentação',
    'atestado', 'protocolo', 'rematrícula', 'registro acadêmico',
    'presença', 'nota', 'cancelamento', 'trancamento',
    'segunda via', 'mudança de curso', 'mudança de polo', 'comprovante'
  ];

  if (palavrasSecretaria.some(p => t.includes(p))) {
    return 'secretaria';
  }

  // ... regras para suporte, comercial, cobrança

  return 'comercial'; // fallback
}
