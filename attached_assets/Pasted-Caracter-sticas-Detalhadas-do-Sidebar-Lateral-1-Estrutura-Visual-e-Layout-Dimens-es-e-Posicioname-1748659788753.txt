Características Detalhadas do Sidebar Lateral
1. Estrutura Visual e Layout
Dimensões e Posicionamento:
Largura: md:w-64 (256px em telas médias e maiores)
Posição: flex-shrink-0 (não diminui de tamanho)
Responsividade: Em mobile vira coluna completa (flex-col md:flex-row)
Posicionamento: sticky top-6 (fica fixo ao rolar a página)
Container Principal:
.bg-card border rounded-md p-4 sticky top-6
Fundo branco/escuro conforme tema
Borda sutil
Cantos arredondados
Padding interno de 16px
Permanece visível ao rolar
2. Barra de Busca
Características:
Input de pesquisa: Campo de texto no topo
Placeholder: "Buscar configurações..."
Estilo: text-sm (texto pequeno)
Funcionalidade: Filtro em tempo real das opções
<Input placeholder="Buscar configurações..." className="text-sm" />

4. Elementos de Navegação
Estrutura de Cada Item:
<Button 
  variant="ghost"                    // Estilo padrão (transparente)
  className="w-full justify-start text-sm"  // Largura total, alinhado à esquerda
  asChild                           // Permite usar como Link
>
  <Link href={item.href}>
    <span className="mr-2">{item.icon}</span>  // Ícone com margem
    {item.label}                              // Texto do item
  </Link>
</Button>
Variantes de Estado:
Estado Normal: variant="ghost" (cinza claro ao hover)
Estado Ativo: variant="default" (azul, texto branco)
Espaçamento: mr-2 entre ícone e texto (8px)

6. Espaçamento e Hierarquia
Estrutura de Espaçamento:
/* Container principal */
.space-y-6          /* 24px entre seções principais */
/* Dentro de cada seção */
.space-y-1          /* 4px entre itens de menu */
.mb-2               /* 8px abaixo do título da seção */
.mb-4               /* 16px abaixo da barra de busca */
Hierarquia Tipográfica:
Título da seção: text-sm font-medium (14px, negrito)
Item de menu: text-sm (14px, peso normal)
Barra de busca: text-sm (14px)
7. Interatividade e Estados
Estados dos Botões:
// Estado padrão
variant="ghost"
className="w-full justify-start text-sm"
// Estado ativo (exemplo: IA - Prof. Ana)
variant="default" 
className="w-full justify-start text-sm bg-primary text-primary-foreground"
// Hover states (automático via Tailwind)
hover:bg-accent hover:text-accent-foreground
Navegação:
Roteamento: Utiliza wouter para navegação SPA
Links: Componente Link do wouter
Atributo asChild: Permite que Button funcione como Link
8. Responsividade
Breakpoints:
/* Mobile (padrão) */
.flex-col           /* Menu em coluna */
/* Tablet e Desktop (md+) */
.md:flex-row        /* Layout em linha */
.md:w-64           /* Largura fixa do sidebar */
Comportamento Mobile:
Menu vira header horizontal
Itens podem quebrar em múltiplas linhas
Sticky behavior pode ser ajustado
